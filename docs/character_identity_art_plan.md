# Hollowfen Character Visual Identity Art Plan

This document is the production queue for dedicated character identity sheets. The HTML storybook now has a `Character Visual Identity Cards` section that uses current story-card art as continuity references and displays the generated model sheets as the primary image. Final identity sheets are saved under:

`public/concept/character-identity/<character-id>-identity-card.png`

Recommended sheet format for each character:

- Main three-quarter portrait matching the existing story-card likeness.
- Front, side, and back full-body turnaround on a neutral parchment/atelier background.
- Two expression or posture studies.
- Prop callouts with scale notes.
- Costume material swatches.
- No readable text baked into the image unless added manually later in HTML.

## Shared Prompt Base

Use case: stylized-concept
Asset type: game character visual identity card / model sheet
Primary request: Create a grounded painterly-medieval character identity sheet for Hollowfen - The Failing Village.
Style: realistic painterly concept art, cinematic but practical, rural late-medieval fantasy with no overt magic, natural materials, muted earth palette, handcrafted clothing, weathered faces, no glamour posing.
Layout: one large three-quarter portrait, front/side/back full-body turnaround, two small expression or posture studies, and prop callouts arranged as a clean art-book production sheet on warm parchment. Do not include readable text, logos, UI, labels, or watermarks.
Continuity: match the character's established story-card likeness, costume, age, and emotional tone.
Avoid: anime, cartoon, glossy fantasy armor, modern clothes, bright saturated colors, heroic combat pose, perfect skin, ornate royal costume unless the character is Aldric.
