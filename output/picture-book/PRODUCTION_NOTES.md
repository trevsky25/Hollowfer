# Hollowfen Picture Book - Production Notes

## Final artifact

- PDF: `output/pdf/hollowfen-picture-book-print-ready.pdf`
- Page count: 40 pages
- Trim size: 10 x 8 inches, landscape
- Bleed: 0.125 inches on all four edges
- PDF media size: 10.25 x 8.25 inches (738 x 594 points)
- Color: CMYK photographic assets with Coated GRACoL 2006 output intent
- Raster preparation: 300 ppi, CMYK JPEG, quality 90
- Type: embedded, subsetted Georgia Regular, Bold, and Italic
- Page boxes: MediaBox and BleedBox at bleed size; TrimBox and ArtBox at finished trim size
- Narrative: full narrative passages for Acts II-IV; picture-book abridgments of the longer Act I passages

This is a printer-neutral production PDF. Confirm the chosen printer's requirements before upload; some print-on-demand vendors request RGB files, omit output intents, or use a different bleed convention.
